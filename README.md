# Витрина магазина приложений

## Обязательные требования
### Выполнено
- **Онбординг**: показывается только при первом запуске. Содержит логотип, приветственный текст и кнопку перехода на витрину приложений.
- **Витрина приложений**: список приложений, где у каждого элемента есть иконка, название, краткое описание и категория. При клике открывается карточка приложения. Присутствует кнопка перехода к списку категорий.
- **Карточка приложения**: содержит кнопку «Назад», иконку, название, название компании-разработчика, категорию, возрастной рейтинг, кнопку «Установить», горизонтальный список скриншотов (при клике открывается экран полноэкранных скриншотов) и полное описание приложения.

## Опциональные требования
### Выполнено
- **Список категорий**: экран содержит название категории и количество приложений в ней. При клике на категорию открывается витрина приложений, отфильтрованная по выбранной категории.
- **Экран полноэкранных скриншотов**: открывается при клике на скриншот из карточки приложения (открывается именно тот скриншот, на который кликнули). Содержит кнопку «Назад», возвращающую в карточку, и горизонтальный список скриншотов.
